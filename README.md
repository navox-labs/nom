# nom - Crab Cookie Clicker

A browser-based Cookie Clicker game with Atari pixel art vibes where crabs eat cookies.

## Play

Open `index.html` in any modern browser. That's it. No server, no install, no dependencies.

## Features

- **Click the cookie** to earn cookies
- **Hire crabs** that passively generate cookies per second
- **10 upgrade tiers** from Baby Crab to Crab God
- **20 achievements** with crab puns
- **Prestige system** — "Molt" to reset for permanent multipliers
- **Offline earnings** — your crabs work while you're away
- **Auto-save** to localStorage
- **CSS pixel art** — no image files, pure box-shadow sprites
- **Retro Atari aesthetic** with CRT scanline effect
- **Responsive** — works on desktop and mobile

## Tech

- Single HTML file (~1300 lines)
- Vanilla JS, no frameworks
- CSS pixel art via box-shadow
- localStorage for persistence
- requestAnimationFrame game loop
